Hardware en detalle

CPU

  • Núcleos e hilos: más paralelismo para multitarea y render.
  • Frecuencia (GHz): influye en la latencia y tareas monohilo.
  • Cache L1/L2/L3: reduce accesos a memoria más lenta.
  • Socket y chipset: compatibilidad con la placa base.
  • Consumo (TDP): impacta en la refrigeración y la fuente.

GPU

  • Shaders/CUDA/Stream: capacidad de cómputo gráfico y GPGPU.
  • VRAM: texturas y modelos a alta resolución sin cuellos de botella.
  • Ancho de banda: bus y tipo de memoria (GDDR6, HBM).
  • Soporte API: DirectX, Vulkan, OpenCL, CUDA.
  • Consumo y tamaño: requisito para la caja y la fuente.

Almacenamiento

Característica SSD NVMe HDD
Velocidad Muy alta (PCIe, GB/s) Moderada (MB/s)
Latencia Baja Alta
Capacidad €/GB Más cara Más barata
Ruido/Vibración Prácticamente nulo Perceptible
Uso recomendado Sistema/juegos/edición Almacén masivo

Para un equipo equilibrado: SSD NVMe para sistema y aplicaciones; HDD opcional para archivos grandes.

Fuente de alimentación (PSU)

  • Potencia real (W) con margen del 20–30% sobre el pico esperado.
  • Certificación 80 Plus (Bronze/Gold/Platinum) para eficiencia.
  • Railes y conectores PCIe/CPU suficientes para la GPU/CPU elegidas.
  • Formato (ATX/SFX) según el chasis.

Refrigeración

  • Disipador aire o líquida AIO según TDP y espacio disponible.
  • Curvas de ventilador ajustadas para equilibrio ruido/temperatura.
  • Pasta térmica aplicada de forma uniforme.

Caja y flujo de aire

  • Entradas frontales/bajo y salidas traseras/superiores para flujo.
  • Filtros antipolvo y gestión de cables para mejorar temperaturas.
  • Compatibilidad con longitudes de GPU y altura de disipador.

Periféricos

  • Monitor: resolución, tasa de refresco (Hz) y tipo de panel.
  • Teclado y ratón según el uso (ofimática, juego, diseño).
  • Audio: interfaz USB o DAC dedicado si se requiere baja latencia.

Especificaciones clave

  • CPU: núcleos/hilos, frecuencia, caché, TDP.
  • GPU: VRAM, ancho de banda, consumo.
  • RAM: 16–32 GB DDR4/DDR5 según el caso.
  • SSD: NVMe PCIe para el SO y apps.
  • PSU: 80 Plus, conectores suficientes.

Glosario rápido

TDP
Potencia térmica a disipar.
VRAM
Memoria dedicada de la GPU.
PCIe
Interfaz de alta velocidad para GPU/SSD.
NVMe
Protocolo para SSDs de baja latencia.

Consejos de compra

  • Equilibra CPU y GPU según el uso principal.
  • Prioriza SSD rápido; amplía almacenamiento después.
  • Deja margen en la PSU para futuras ampliaciones.

Guía de diseño — 6 puntos

  1. Coherencia visual: paleta 3–5 colores, 1–2 tipografías y estilo uniforme.
  2. Jerarquía clara: destaca lo importante con tamaño, color y posición.
  3. Espacios en blanco: márgenes y paddings para evitar saturación visual.
  4. Navegación intuitiva: menús claros también en móviles.
  5. Optimización y accesibilidad: contraste, ALT, teclado y carga rápida.
  6. Elementos destacados: botones visibles y microinteracciones suaves.
